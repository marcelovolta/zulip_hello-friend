# Hello, Friend - Política de privacidad

**1) Quiénes somos**
Este servidor de Zulip es una comunidad independiente, auto-alojada (“self-hosted”) en infraestructura propia. El/los administradores del servidor actúan como responsables del funcionamiento y la moderación.

**2) Qué datos recopilamos**
Según el uso normal de Zulip, podemos almacenar:

* datos de cuenta: nombre para mostrar, email, foto/avatar (si la subís), configuración de usuario;
* contenido generado por usuarios: mensajes, archivos adjuntos que se suban al servidor, reacciones, ediciones;
* datos técnicos básicos: registros (logs) del servidor que pueden incluir direcciones IP, user agent, timestamps y eventos de acceso para fines de seguridad y diagnóstico.

**3) Para qué usamos los datos**
Usamos estos datos para:

* operar la plataforma y permitir la participación en la comunidad;
* moderar y hacer cumplir las normas;
* mantener la seguridad, prevenir abuso y resolver incidentes;
* realizar copias de seguridad y mantenimiento.

**4) Base de funcionamiento y consentimiento**
Al crear una cuenta y usar el servidor, aceptás que tus mensajes y acciones dentro de Zulip sean almacenados y procesados con el único fin de operar la comunidad.

**5) Compartición con terceros**
No vendemos ni cedemos datos. Podríamos usar servicios externos únicamente para funciones técnicas, por ejemplo:

* **SMTP** para envío de correos (invitaciones, restablecimiento de contraseña, notificaciones), si estuviera habilitado.
  En ese caso, el proveedor de email podría procesar metadatos (destinatario, asunto, timestamps).
  

**6) Retención**
Conservamos los datos mientras la cuenta exista o mientras sea necesario para operar el servidor. Los backups pueden retener datos durante un período adicional.

**7) Derechos y solicitudes**
Podés solicitar:
* acceso a tus datos de cuenta,
* corrección de información,
* eliminación de cuenta (tené en cuenta que puede permanecer contenido en backups por un tiempo).
  Contacto: **[email o canal de admin]**.

**8) Seguridad**
Aplicamos medidas razonables (TLS/HTTPS, actualizaciones, control de acceso, backups) para proteger la información. Ver también las guías generales de seguridad y cumplimiento de Zulip. ([Zulip][2])

**9) Cambios**
